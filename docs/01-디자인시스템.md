## 네비게이션 바
- 그라디언트: `linear-gradient(135deg, #b8c0ff → #d4bbfc)`
- 텍스트: `#4a4a6a` (진한 보라 회색)

## 카드 스타일
- 테두리: 없음
- 모서리: `16px` (기존 12px에서 확대)
- 그림자: `0 2px 8px rgba(0,0,0,.04)`
- 호버: `translateY(-3px)` + `box-shadow 0 6px 20px rgba(0,0,0,.06)`

## 차트 설정
- 바 차트 모서리: `borderRadius: 6`
- 도넛 차트 컷아웃: `65%`
- 라인 차트 텐션: `0.4`
- 그리드 색상: `#f0ecff`
- 범례: `usePointStyle: true, pointStyle: 'circle'`
