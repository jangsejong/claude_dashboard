# 02. 프론트엔드 변경사항

## 변경 파일
- `backend/static/index.html`

## 변경 내용

### CSS 변경
| 항목 | 기존 | 변경 |
|------|------|------|
| 배경 | `#f0f2f5` (회색) | `#f5f3ff` (라벤더 화이트) |
| 네비바 | `#667eea → #764ba2` (진한 보라) | `#b8c0ff → #d4bbfc` (파스텔 보라) |
| 네비바 텍스트 | 흰색 | `#4a4a6a` (진한 보라회색) |
| 카드 모서리 | `12px` | `16px` |
| 카드 그림자 | `0 1px 3px rgba(0,0,0,.08)` | `0 2px 8px rgba(0,0,0,.04)` |
| 호버 효과 | `translateY(-2px)` | `translateY(-3px)` + 부드러운 그림자 |
| 테이블 호버 | Bootstrap 기본 | `#f8f5ff` (연보라) |
| code 태그 | Bootstrap 기본 | `배경 #f0edff, 색상 #6b5eaa` |

### KPI 카드
- 아이콘+배경 원형 배지 추가 (`.kpi-icon`)
- Bootstrap 기본 색상 클래스(`text-primary` 등) → 파스텔 인라인 색상

### 차트 개선
- `USER_COLORS`: 8색 전부 파스텔톤으로 교체
- 바 차트: `borderRadius: 4 → 6`, 그리드 `#f0ecff`
- 도넛 차트: `cutout: 65%`, `borderWidth: 0`
- 라인 차트: `tension: 0.3 → 0.4`, `pointRadius: 2 → 3`
- 범례: 원형 포인트 스타일 통일

### 한글화
- 영문 주석 → 한국어 (`// Auto-refresh every 60 seconds` → `// 60초마다 자동 갱신`)
- `console.error` 메시지 한국어화
- 하단 푸터 텍스트 추가
